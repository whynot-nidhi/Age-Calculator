# Age Calculator

A web-based Age Calculator built using HTML, CSS, and JavaScript that calculates the exact age between two dates and displays the result in years, months, and days.

## Features

- Calculate age between any two selected dates
- Display age in years, months, and days
- Zodiac sign detection based on birth date
- Simple and user-friendly interface
- Responsive design for different screen sizes

## Tech Stack

- HTML5
- CSS3
- JavaScript
